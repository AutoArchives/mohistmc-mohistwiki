## Setup Mohist for development

> Available versions : **1.7.10**, **1.12.2**, **1.16.5**, **1.18.2**, **1.19.2**, **1.19.4**, **1.20**
### Steps
* Checkout project
    ```bash
    git clone https://github.com/MohistMC/Mohist.git -b <mc-version>
    ```
* Setup
  * Setup with Linux:  
    `bash gradlew setup`
  * Setup with Windows:  
    `gradlew.bat setup`
